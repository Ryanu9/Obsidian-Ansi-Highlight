# Ansi Highlight

一个可以用显示ANSI高亮的obsidian插件，适合用于记录终端高亮输出。推荐搭配 https://github.com/Ryanu9/ansi_copy 使用

![1770488144451](image/README/1770488144451.gif)

## 安装:

- 将 `main.js`、`styles.css`、`manifest.json` 复制到你的obsidian  `plugins/ 目录`中。


## 使用:

- 创建ansi代码块，复制ansi源码到ansi代码块中即可
- ctrl+鼠标左键点击代码块区域 可以进入编辑模式
- 代码块右上角可以切换ansi源码模式和渲染模式
